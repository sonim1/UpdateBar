# Security Blockers

There are no open security-configuration blockers as of 2026-07-20.

GitHub branch protection is enabled for `main` with strict `macos` and `linux`
status checks, one approving pull-request review, CODEOWNERS review, stale-review
dismissal, conversation resolution, and force-push/deletion prevention. Repository
administrators retain bypass permission so the solo-maintainer repository cannot
be locked by GitHub's prohibition on self-approval.
